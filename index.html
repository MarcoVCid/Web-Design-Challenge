Summary: Latitude vs. X

The purpose of this project was to analyse how weather changes as you get closer to the Equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities


After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at, included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.

Max Temperature

As expected, the weather becomes significantly warmer as one approaches the equator (0 Deg. Latitude). More interestingly, however, is the fact that the southern hemisphere tends to be warmer this time of year than the northern hemisphere. This may be due to the tilt of the earth at the time of the year this data was gathered.


Humidity

We found no comparison between Humidity and Latitude. As seen in the graph, there are many random numbers because there is no correlation between this two variables.


Cloudiness

The Cloudiness reacts just like the Humidity but in the inverse way. We can not find any correlation between both variables. Maybe a better way to analyze this variable would be by comparing it against wind speed.


Wind Speed

There is a strange correlation between this two variables. To start with, it may seem that there is no correlation between them. If we take some time to analyze it, we can see that the higher the latitude the higher is the Wind Speed.
